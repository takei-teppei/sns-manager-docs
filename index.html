Features

3つの主要機能

SNS運用で時間を取られる作業を、すべて自動化します

📅

Instagram
予約投稿

投稿内容・画像・ハッシュタグを入力して日時を設定するだけ。あとはすべて自動で投稿されます。

  • カルーセル投稿（複数画像）に対応
  • Google Drive の画像をそのまま使用
  • 5分間隔で予約をチェック・自動実行
  • 投稿結果をシートに自動記録
🔗

他SNSへの
連携投稿

Instagramと同じ内容を、チェックボックスひとつでX・TikTok・Threadsにも同時投稿。

  • X (Twitter) — OAuth 1.0a対応
  • Threads — Meta APIで高品質連携
  • TikTok — 動画ファイル対応
  • プラットフォームごとの文字数自動調整
📊

分析・
レポート機能

各SNSのエンゲージメントデータを自動収集。ワンクリックでサマリーレポートを生成します。

  • いいね・コメント・リーチ・保存数
  • エンゲージメント率の自動計算
  • 月別・プラットフォーム別の集計
  • レポートシートを自動生成

FAQ

よくある質問

プログラミングの知識は必要ですか？ +
いいえ、不要です。Googleスプレッドシートが使えれば問題ありません。初期セットアップはコピー&ペーストで完了します。APIキーの取得についてはセットアップガイドに詳細な手順を記載しています。
月額費用は本当にかかりませんか？ +
はい、SNS Manager本体のランニングコストは¥0です。Google Apps ScriptとGitHub Pagesは無料の範囲内で動作します。ただしX(Twitter)の投稿機能はAPIのBasicプラン（月額$100）が必要です。Instagramのみご利用の場合は追加費用は一切かかりません。
買い切り版は将来のアップデートも受けられますか？ +
はい。同じメジャーバージョン（例：v1.x.x）内のアップデートはすべて無料でご利用いただけます。メジャーバージョンアップ（v2.0など）の場合は優待価格でご提供する予定です。
複数のInstagramアカウントで使えますか？ +
買い切りプラン（Starter・Standard・Pro）は1ライセンス = 1スプレッドシート（1アカウント）でのご利用となります。2〜3アカウントをお使いの場合はTeamプラン（¥98,000・3アカウントまで）、4アカウント以上または法人での複数人利用はBusinessプラン（月額¥4,980）が最もお得です。
社内の複数メンバーで1ライセンスを共有できますか？ +
申し訳ございませんが、利用規約により1ライセンスの複数スプレッドシートへの複製・配布は禁止しております。スプレッドシートIDをライセンスキーに紐付けているため、技術的にも別シートでの利用はブロックされます。複数メンバーで使用される場合はTeamプランまたはBusinessプランをご検討ください。
投稿に失敗した場合はどうなりますか？ +
投稿に失敗した場合、ステータスが「エラー」に変わり、エラー内容がシートに記録されます。Pro・Businessプランではエラー発生時にLINEまたはメールで即時通知します。
サポートはどのように受けられますか？ +
LINE公式アカウントにてサポートを受け付けています。Businessプランは優先対応（通常24時間以内に返信）、その他のプランは通常3営業日以内の対応となります。
