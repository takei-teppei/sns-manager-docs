💳料金・購入
買い切りと月額（Business）の違いは何ですか？ +
項目買い切り（Pro等）Business（月額）
支払い一度きり毎月 ¥4,980
アカウント数1〜3（プランによる）無制限
サポート通常対応優先対応（翌営業日）
新機能同メジャーバージョン内は無料常に最新
複数アカウントを管理する代理店・法人の方はBusinessプランがお得です。個人・1アカウントの方は買い切りが長期的にはコストを抑えられます。
購入後の返金はできますか？ +
購入から7日以内かつ未使用（未アクティベート）の場合に限り、返金対応いたします。ライセンス認証（アクティベート）済みの場合は原則として返金対応が難しくなりますので、購入前にご不明点はLINEサポートにてご確認ください。
領収書・インボイスは発行できますか？ +
はい、発行可能です。購入後にLINEサポートへ「領収書希望」とご連絡いただき、宛名・日付をお知らせください。PDF形式にてお送りします。
月額SaaSと比較して本当にお得ですか？ +
はい。一般的なSNS管理SaaSは月額¥3,000〜¥10,000程度かかります。月額¥5,000とすると年間¥60,000。SNS Manager Proは¥69,800の買い切りなので、約14ヶ月で元が取れ、それ以降はずっと¥0で使い続けられます。さらにサーバー代・インフラ代も一切かかりません。
🔑ライセンス
1ライセンスで何アカウント使えますか？ +
プランによって異なります。

Starter / Standard / Pro：1ライセンス = 1スプレッドシート（1アカウント）専用
Team：3スプレッドシートまで使用可（追加 +¥20,000/件）
Business：アカウント数無制限

ライセンスキーはスプレッドシートIDに紐付けて管理されているため、別のスプレッドシートにコピーしての利用は技術的にもブロックされます。
社内の複数メンバーで1ライセンスを共有・使い回せますか？ +
できません。利用規約により、1ライセンスを複数スプレッドシートに複製・配布することは禁止しております。複数名での利用にはTeamプランまたはBusinessプランをご選択ください。不正利用が検知された場合はライセンスを無効化させていただきます。
スプレッドシートを作り直したい場合、再認証できますか？ +
はい、可能です。LINEサポートへ「スロットリセット希望」とご連絡いただければ、担当者が紐付けをリセットし、新しいスプレッドシートで再アクティベートできるようにします。

スロットリセット対応は月1回まで無料です。
将来のアップデートは無料ですか？ +
同じメジャーバージョン（例：v1.x.x）内のアップデートはすべて無料です。メジャーバージョンアップ（v1 → v2）の際は、既存ユーザー向けに優待価格（通常の50%オフ予定）を設ける予定です。Businessプランはバージョンに関わらず常に最新版をご利用いただけます。
⚙️セットアップ
プログラミングの知識は必要ですか？ +
不要です。Googleスプレッドシートの基本操作ができれば使えます。初期設定はAPIキーのコピー&ペーストのみです。APIキーの取得手順はセットアップガイドで画像付きで解説しています。
セットアップにどのくらい時間がかかりますか？ +
Instagramのみ（Starterプラン）であれば30〜60分程度でセットアップ完了です。全SNS（Standard・Pro）の場合は各SNSのAPI申請が含まれるため2〜3時間程度かかります。特にTikTokとXは審査が必要なため、申請から承認まで数日かかる場合があります。
X（Twitter）のAPIはどのプランが必要ですか？有料ですか？ +
X APIのBasicプランが必要です（月額$100 / 約¥15,000）。これはXが課金する費用であり、SNS Manager本体の料金とは別途かかります。Xへの投稿が不要な場合はFreeティアでは投稿APIが利用できないため、X連携機能はご利用いただけません。Instagramのみ利用する場合は追加費用は一切かかりません。
Instagramはビジネスアカウントが必要ですか？ +
はい、必須です。Instagram Graph APIはビジネスアカウントまたはクリエイターアカウントのみ利用できます。個人アカウントをプロアカウントに切り替える手順はInstagramアプリの設定から無料で行えます。
📝投稿機能
Instagramのカルーセル投稿（複数画像）に対応していますか？ +
はい、対応しています。投稿管理シートのB列（画像URL）にGoogle DriveのURLをカンマ区切りで複数入力すると、自動的にカルーセル投稿になります。

例：https://drive.google.com/file/d/AAA…, https://drive.google.com/file/d/BBB…
予約投稿の精度（タイミングのズレ）はどのくらいですか？ +
スケジューラーは5分ごとに動作するため、最大5分のズレが生じます。例えば14:00に予約した投稿は14:00〜14:05の間に投稿されます。秒単位の精度は保証できませんが、通常の運用では問題ないレベルです。
動画（リール）投稿に対応していますか？ +
TikTokは動画投稿のみ対応しています。Instagram・X・Threadsはv1では画像投稿が主対応で、リール（動画）投稿はv1.2以降で対応予定です。アップデート通知はサポートLINEにてご案内します。
投稿に失敗した場合はどうなりますか？自動でリトライされますか？ +
投稿失敗時はステータスが「エラー」に変わり、I列にエラー内容が記録されます。Pro・Businessプランではエラー発生と同時にLINEまたはメールで通知します。

自動リトライは行いません（誤投稿防止のため）。エラー内容を確認・修正してからステータスを「予約済み」に戻すと再投稿できます。
📊分析機能
分析データはリアルタイムで取得できますか？ +
リアルタイムではなく、手動または定期取得です。「📱 SNS Manager」→「分析データを取得」を実行するとその時点のデータが記録されます。自動取得はメニューからトリガーを設定すると毎日指定時刻に自動実行できます。
分析機能はどのプランから使えますか？ +
Proプラン以上でご利用いただけます。Starter・Standardプランは投稿・予約のみです。後からProプランに買い替えた場合も同じスプレッドシートでアップグレード適用できます（サポートまでご連絡ください）。
🔧トラブル
予約時刻を過ぎても投稿されません。 +
以下を順番に確認してください。

1. トリガーが設定されているか
「📱 SNS Manager」→「トリガーを設定」を再実行してください。

2. ステータスが「予約済み」になっているか
H列が「下書き」のままでは投稿されません。

3. 「ログ」シートにエラーが記録されていないか
APIキーの有効期限切れ・権限不足などが原因のことがあります。

解決しない場合はログシートのエラー内容をLINEサポートにお送りください。
「アクセストークンが無効」というエラーが出ます。 +
Instagramのアクセストークンは通常60日で期限が切れますが、SNS Managerは30日ごとに自動更新します。それでもエラーが出る場合は、Meta Developersで新しいトークンを発行し直し、設定シートのB3に貼り替えてください。手順はセットアップガイドを参照してください。
「権限が必要です」というエラーが出て操作できません。 +
GASの実行には初回に権限の付与が必要です。

1. メニューから任意の操作を実行
2. ダイアログが表示されたら「権限を確認」をクリック
3. Googleアカウントを選択
4. 「詳細」→「（プロジェクト名）に移動（安全ではないページ）」をクリック
5. 「許可」をクリック

※「安全ではない」と表示されますが、これはGoogleが外部に公開されていないGASに表示する標準の警告です。SNS Manager本体のコードに問題はありません。
「別のスプレッドシートに紐付けられています」というエラーが出ます。 +
このメッセージはライセンスキーが別のスプレッドシートに登録済みであることを示しています。

スプレッドシートを作り直した場合：LINEサポートへ「スロットリセット希望」とご連絡ください。旧シートのIDを解除し、新しいシートで再認証できるようにします。

身に覚えがない場合：第三者によるキーの不正利用が疑われます。同様にLINEサポートにご連絡ください。
